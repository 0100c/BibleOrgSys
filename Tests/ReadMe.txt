ReadMe.txt for BibleOrgSys/Tests/

The test modules in this folder are intended to be run from the containing folder.

In other words, from the BibleOrgSys folder, run a test module like:
    cd ..
    Tests/BibleBooksCodesTest.py
        or      python3 Tests/BibleBooksCodesTest.py

or to run all tests
    cd ..
    Tests/TestSuite.py
        or      python3 Tests/TestSuite.py
